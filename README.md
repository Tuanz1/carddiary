# 项目说明
Card Diary是一款日记软件，界面美观（UI灵感来源于Behance上的一个CardDiary项目）。由于本人需求，所以自己实现了3端版本,并且集成Markdown支持，免费提供给所有想要记日记的人。

## 项目架构
Ionic5/Angular + Parse
